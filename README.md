# Шулепов Иван Ит-2 Лабораторная №6

## Задание 1

### Задача 1

**Текст задачи**

@Invoke.  
Разработайте аннотацию @Invoke со следующими характеристиками:  
• Целью может быть только МЕТОД  
• Доступна во время выполнения программы  
• Не имеет свойств  

Создайте класс, содержащий несколько методов, и отметьте хотя бы один из них аннотацией @Invoke.  
Реализуйте обработчик (с помощью Reflection API), который находит методы, помеченные аннотацией @Invoke, и автоматически вызывает их.

**Алгоритм решения**

Создание аннотации @Invoke для автоматического вызова метода без свойств. Создание демонстрационного класса с различными методами, проверка работы аннотации. Создание обработчика аннотации, который: проверяет, что объект не пустой, извлекает все методы класса, проверяет наличие аннотации у каждого метода и, если она есть, применяет ее.

### Задача 3

**Текст задачи**

@ToString.  
Разработайте аннотацию @ToString со следующими характеристиками:  
• Целью может быть ТИП или ПОЛЕ  
• Доступна во время исполнения программы  
• Имеет необязательное свойство value с двумя вариантами значений: YES или NO  
• Значение свойства по умолчанию: YES  

Проаннотируйте класс аннотацией @ToString, а одно из полей – с @ToString(Mode.NO).  
Создайте метод, который формирует строковое представление объекта, учитывая только те поля, где @ToString имеет значение YES.

**Алгоритм решения**

Создание аннотации @ToString для формирования строкового представления объекта, учитывая поля YES. Создание демонстрационного класса с методами, проверка работы аннотации. Создание обработчика аннотации, который: создаёт StringBuilder для построения строки, получает все поля от объекта, для каждого поля получает аннотацию, если значение аннотации Mode.YES, то добавляет имя поля и значение в StringBuilder.
